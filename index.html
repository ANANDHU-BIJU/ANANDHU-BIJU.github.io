Jan 2026 – Mar 2026 · Mini Project
Human-Centric Embodied Home Assistant
🏆 First Prize — INNOVIX / RISE 2026 at FISAT

Overview

A Human-Centric Embodied Home Assistant designed to enhance human–machine interaction in smart home environments. Unlike conventional assistants that rely solely on voice, this system integrates computer vision, speech interaction, and embedded control — enabling face recognition, presence detection, and expressive physical interaction through a Raspberry Pi 2 Model B and ESP32 controller.

Key Features

  • Face recognition for identity-based secure interaction
  • Voice interaction via USB microphone with speech-to-text and text-to-speech
  • Wake word navigation — layout-driven movement to predefined rooms when no face is detected
  • 3-DOF servo head movement for expressive, user-oriented physical interaction
  • Radar-based human presence detection for improved responsiveness
  • TFT display for visual feedback and expressions
  • Wireless home automation via ESP8266 — WiFi relay control for appliances
  • Web-based remote inspection — motor control and image capture via browser
  • Custom PCB design for improved reliability and compactness
  • Battery-powered with regulated supply rails for portable operation

2026 · Hardware Design
Custom PCB — Home Assistant System

Overview

A custom PCB designed using KiCad to integrate the core hardware components of the Home Assistant system into a single compact unit. The board consolidates connections for the Raspberry Pi, ESP32 DevKit, servo motors, radar sensor (LD2410), motor driver (TB6612FNG), TFT display, and power supply components — simplifying wiring and significantly improving system reliability. Manufactured and ordered through PCBWay.

Key Features

  • Integrates Raspberry Pi, ESP32, servo motors, radar sensor, and motor driver on a single board
  • TB6612FNG motor driver for precise and efficient motor control
  • LD2410 radar sensor integration for human presence detection
  • Power regulation and distribution for both high-power (motors) and low-power (logic) components
  • Organized, stable connections replacing complex wiring harnesses
  • Compact form factor improving system enclosure and portability
  • Manufactured and delivered via PCBWay
